# STP 战略详解

#tags: #市场营销 #STP #战略规划

## 概述

STP 是现代营销战略的核心框架，由三个关键步骤组成：
- **S**egmentation（市场细分）
- **T**argeting（目标市场选择）
- **P**ositioning（市场定位）

## 步骤一：市场细分

### 有效的市场细分标准

| 标准 | 说明 |
|------|------|
| 可衡量性 | 细分市场的规模和购买力可衡量 |
| 可进入性 | 细分市场可以被有效触及和服务 |
| 可盈利性 | 细分市场足够大，有利可图 |
| 可行动性 | 企业有资源在该细分市场开展活动 |

### 常用细分变量

- 地理细分
- 人口细分
- 心理细分
- 行为细分

## 步骤二：目标市场选择

### 评估标准

1. 细分市场的规模和增长率
2. 细分市场的结构性吸引力
3. 企业目标和资源的一致性

### 五种选择模式

1. **单一细分市场**：集中资源服务一个细分市场
2. **选择性专门化**：选择几个互不相关的细分市场
3. **产品专门化**：向各类顾客提供一类产品
4. **市场专门化**：为某个特定顾客群提供多种产品
5. **完全覆盖**：服务所有顾客群体

## 步骤三：市场定位

### 定位流程

1. 识别目标顾客感知的关键价值维度
2. 分析竞争对手在这些维度的位置
3. 选择差异化的定位点
4. 传播和强化定位

### 定位陈述模板

> "针对 [目标细分市场]，[品牌名称] 是 [产品品类] 中能够 [关键利益点] 的品牌，因为 [独特证据]。"
