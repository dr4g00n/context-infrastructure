## 1. 产品概述

### 1.1 基本信息

| 属性 | 详情 |
|------|------|
| **产品名称** | Flipper Zero |
| **产品定位** | 便携式多功能安全测试工具 |
| **目标用户** | 安全研究员、渗透测试人员、电子爱好者 |

| **外形设计** | Tamagotchi 风格，海豚主题 |
| **开源状态** | ✅ 固件完全开源 |

### 1.2 产品愿景

Flipper Zero 是一款为渗透测试人员和硬件爱好者设计的便携式多工具设备。它可以帮助用户研究和分析无线通信系统、测试访问控制系统、以及探索日常设备如何通信。

## 2. 硬件规格

| **红外** | IR 收发 | 红外遥控 |
| **iButton** | 1-Wire | 接触式通信 |

| **WiFi** | 需扩展模块 | WiFi 功能 |

### 2.3 扩展能力

## 3. 核心功能详解

### 3.1 RFID/NFC 功能

#### 低频 (LF) RFID - 125 kHz
- **读取**：读取 EM4100 等低频卡
- **模拟**：模拟已读取的卡片
- **写入**：部分卡支持写入（如 T5577）

#### 高频 (HF) NFC - 13.56 MHz
- **读取**：Mifare Classic、NTAG 等
- **模拟**：模拟 NFC 卡片
- **破解**：Mifare Classic 密钥破解（需配合额外工具）
- **应用**：门禁卡、公交卡、支付卡研究

### 3.2 红外 (IR) 遥控功能

| 功能 | 说明 |
|------|------|
| **学习** | 学习遥控器红外信号 |
| **保存** | 保存红外遥控码库 |
| **发送** | 发送红外信号控制设备 |
| **应用** | 电视、空调、音响等家电控制 |

### 3.3 Sub-GHz 无线功能

| 频段 | 应用场景 |
|------|----------|
| **433 MHz** | 车库门、无线门铃、遥控器 |
| **868 MHz** | 欧洲智能家居设备 |
| **915 MHz** | 美国物联网设备 |
| **特点** | 约 50 米有效范围 |

### 3.4 协议分析与调试

- **USB HID 模拟**：模拟键盘、鼠标进行 BadUSB 攻击测试
- **串口通信**：SPI、UART、I2C 协议分析
- **GPIO 控制**：控制外部电路、读取传感器数据

## 9. 法律与伦理考量

### 9.1 合法使用

Flipper Zero 设计初衷是用于**授权的安全测试**：
- ✅ 测试自己拥有的设备
- ✅ 有明确授权的安全审计
- ✅ 教育研究目的
- ✅ 协议分析学习

### 9.2 法律边界

| 行为 | 合法性 |
|------|--------|
| 未经授权访问他人系统 | ❌ 违法 |
